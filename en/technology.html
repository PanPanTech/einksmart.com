Technical FAQ

Why use color E-Ink instead of normal LCD signage?

Color E-Ink is calmer, lower power, and more suitable for long dwell-time spaces where the display should feel like part of the room rather than a bright advertising screen.

Do you claim every preview is a real product photo?

No. Until a project uses verified production photography, visuals are labeled as E-Ink optimized previews. Production shoots can replace the configured assets later.

How does provenance work?

For authorized content, project metadata can include artist authorization, Art ID, edition notes, usage scope, and optional watermarking. This is about traceability, not speculative trading.

Can the platform support multi-device management?

The current public site describes the B2B direction. Device groups, status, scheduling, and project APIs can be scoped as part of partner deployments.
